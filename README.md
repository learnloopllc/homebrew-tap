# LearnLoop Homebrew Tap

Homebrew formulae for [LearnLoop](https://learn-loop.org) CLI tools.

## Install

```bash
# Stable
brew install learnloopllc/tap/glow

# Beta
brew install learnloopllc/tap/glow-beta
```

Both can be installed side by side — stable installs `glow`/`glw`, beta installs `glow-beta`/`glw-beta`.

## Update

```bash
brew update && brew upgrade glow
brew update && brew upgrade glow-beta
```
